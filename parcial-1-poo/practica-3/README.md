En esta practica 3, utilizamos varios conceptos, como es la encapsulacion, herencia y manejo
de errores.

Lo que hace este sistema, es manejra dos tipos de usuarios, Admin y Alumno, utilizando la herencia de Usuario, que es la base del sistema. 
Al final en el index hace lo mismo que las practicas
anteriores, muestra en pantalla los datos, pero con un cambio, ahora usamos el try/catch, para poder
utilizar excepciones y validar en este caso el correo. Al pasar al catch, se toma el error, y se muestra
un mensaje de error, en lugar de que el programa se detenga, asi se puede seguir ejecutando.
