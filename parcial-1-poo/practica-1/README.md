En esta practica aprendemos conceptos de Programacion Orientada a Objetos (POO), creando una clase, 
instanciandola y encapsulamiento. 

La practica funciona creando un archivo (Usuario.php) donde se crea la clase Usuario,
declarando variables (nombre y correo) y creando las funciones para obtener estos mismos dentro
del archivo index.php.
En este archivo (index.php), incluimos con el include a Usuario.php, instanciamos la clase
dandole parametros de nombre y correo, y por ultimo se mandan llamar con los getters (getNombre
y getCorreo).
